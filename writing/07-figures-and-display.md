# 07 — Figures, tables, captions and artwork

## Role

Display items and their captions must communicate the finding on their own. These rules
apply equally to **main and Supplementary** figures/tables.

## Caption writing

- Open the caption with a short **title phrase** (the figure's message), then describe each
  panel in order: "**a**, [what it shows]. **b**, [what it shows]."
- **Caption ≤ 350 words.**
- **Self-contained:** define *every* acronym, symbol, colour, and abbreviation used in the
  figure **within the caption itself**. The reader must not need the main text. (This is the
  one place you re-define acronyms already defined in the body — required, not a violation.)
- Describe colours/symbols in **plain words** ("blue bars show […]"); do **not** paste
  coloured glyphs into the caption text.
- Significance keys are conventional: "*p* < 0.05, ***p* < 0.01, ****p* < 0.001" (the
  scalar *p* italic).
- No bullet lists in captions (per `00`).

## Composition rules (baked in)

- **No red-green combinations.** Colour-blind readers cannot separate red from green. Use a
  colour-blind-safe palette (e.g. Okabe–Ito, blue–orange, blue–red diverging). If red and
  green are unavoidable, separate them by hue/lightness or add hatching/patterns so they
  differ in greyscale.
- **Maps must show latitude and longitude** labels on the axes (e.g. 60°N…60°S, 120°W…120°E).
- **Multi-panel figures need bold lowercase panel labels** **a, b, c, …** at the upper-left
  of each panel (main *and* Supplementary). Single-panel figures need none.
- **Keys/legends go inside the image**, not only in the caption.

## Counts and fit (baked in)

- **≤ 10 main display items** (figures + tables combined).
- Each display item **fits within one A4 page** (210 × 297 mm).
- Main-text figure legends live **in the manuscript**, not embedded in the figure file; each
  main figure is a **separate file**. **All Supplementary figures live inside the single
  Supplementary PDF** (not uploaded separately).

## Artwork specs (baked in)

- **Width:** single-column ≈ 88 mm; double-column ≈ 180 mm.
- **Fonts:** Arial/Helvetica (sans-serif), **5–7 pt**, text kept **editable** (not
  rasterised).
- **Vector** (AI/EPS/PDF) for charts, schematics, line art, maps, diagrams; **raster ≥ 300
  dpi** for photos/microscopy/3D. Mixed figures: raster base ≥ 300 dpi + editable vector
  overlays (don't flatten).
- **Colour mode:** RGB for original research figures.
- Avoid PowerPoint/Word effects (drop shadow, 3-D, bevel) that degrade on export; export
  Excel/Office charts to PDF, not screenshots.
- Keep each figure file well under the upload limit (target ≪ 50 MB; see `verification/`).

## Supplementary-specific (same rules + these)

- Label prose as **Supplementary Note**, tabular data as **Supplementary Table N**, and
  graphics as **Supplementary Figure N** — never call a table or figure a "Note".
- SI figures/tables follow all caption rules above (≤ 350 words, self-contained, lat/lon,
  panel labels).

## Self-check

Caption ≤ 350 words and self-contained (every acronym/colour defined)? No red-green? Maps
have lat/lon? Panels labelled **a, b, c**? ≤ 10 main items, each on one A4 page? Arial 5–7 pt,
vector where possible, ≥ 300 dpi raster? SI items classified and labelled correctly?
