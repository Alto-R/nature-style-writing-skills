# 09 — Declarations and availability statements

## Role

The required end-matter statements. Each is short but has strict content rules; vague
wording is a common rejection trigger.

## Data availability (baked in)

Under a dedicated **Data availability** heading (in Methods):

- Be **specific and transparent**. **Never** write "available on reasonable request",
  "upon request", or "from the authors".
- Name the exact repository/database, **accession codes**, and **URLs/DOIs** for every
  dataset used or generated.
- If source data underlie the figures, state it: "Source data are provided with this paper."
- If access is genuinely restricted, state the *reason* (privacy, ethics, legal,
  commercial), *what* is restricted, *how* to request it, *who* to contact, and the
  *timeframe*.

Shape: "The [dataset] generated in this study are available in [repository] under accession
[code] (https://doi.org/…). [Third-party dataset] is available at [URL]. Source data are
provided with this paper."

## Code availability (baked in)

Under a dedicated **Code availability** heading:

- If code is on GitHub, mint a **Zenodo DOI** (GitHub→Zenodo) so it is citable.
- Name the repository and **cite the Zenodo DOI**; the reference list carries the full
  citation (authors, title, repository, DOI, year).

Shape: "The code used in this study is available at [GitHub URL] and archived at Zenodo
(https://doi.org/…)."

## Competing interests (baked in)

Under **Competing interests**: either declare specific interests with the relevant **author
initials** (and patent/funding detail), or the explicit negative statement: *"The authors
declare no competing interests."*

## Funding (baked in)

Under **Funding**: disclose all sources, funders, and grant numbers.

## Author contributions (baked in)

Under **Author contributions**:

- Mention **each author by initials** with their specific contribution.
- **Never use "all authors"** as a shortcut.
- Differentiate authors who share initials: "C.H. (Full Name) did …; C.H. (Other Name) did …".

Shape: "A.B. designed the study. A.B. and C.D. performed the analysis. C.D. wrote the
manuscript with input from E.F. E.F. supervised the work."

## Affiliations & corresponding author (baked in)

- Number affiliations sequentially (1, 2, 3…).
- Mark every corresponding author with a symbol (e.g. *) and give the email.
- Mark equal-contribution / joint-supervision authors with a symbol (not a number).

## Self-check

Data availability specific with repositories/accessions/DOIs (no "on request")? Code archived
with a Zenodo DOI? Competing interests stated (or explicitly none)? Funding listed? Author
contributions by initials with no "all authors"? Affiliations numbered, corresponding author
marked with email?
